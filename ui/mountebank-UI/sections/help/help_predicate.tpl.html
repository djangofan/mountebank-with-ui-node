Request Structure

Field Description Type
requestFrom The client socket, primarily used for logging and debugging. string
path The path of the request, without the querystring string
query The querystring of the request object (as key value pairs)
method The request method string
headers The HTTP headers object (as key value pairs)
body The request body string

Note that the Mountebank server must run with the --allowInjection flag. Mountebank Injection documentation
